Format-ArmorApiRequestBody

SYNOPSIS

Generates the JSON request body payload for an Armor API request.

SYNTAX

Format-ArmorApiRequestBody [-Keys] <String[]> [-Parameters] <PSObject[]> [<CommonParameters>]

DESCRIPTION

Retrieves the values of the parameters defined in the parent function that match the names of the specified keys, builds the JSON request body, and then returns the request body payload.

EXAMPLES

EXAMPLE 1

Format-ArmorApiRequestBody -Keys 'key1', 'key2' -Parameters $parameters

Generates a JSON document with the names and values of objects in the $parameters array with names matching key1 & key2.

PARAMETERS

-Keys

Specifies the variables available in the endpoint request body schema.
